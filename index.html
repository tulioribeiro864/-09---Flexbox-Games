Registro de Níveis

Flexbox Zombies

  • Nível 3: - Desafio com múltiplos containers, usei flex-wrap.
  • Nível 5: - Alinhei itens com justify-content.
  • Nível 7: - Ajustei o crescimento de itens com flex-grow.

Flexbox Froggy

  • Nível 4: - Alinhei itens com flex-direction.
  • Nível 6: - Usei align-items para alinhar verticalmente.
  • Nível 8: - Distribuí itens com justify-content (space-between).

Reflexão e Aplicação Prática

  • flex-direction: Define direção (horizontal/vertical). Usei no Flexbox Froggy.
  • justify-content: Alinha itens ao longo do eixo principal. Usado em ambos os jogos.
  • align-items: Alinha itens verticalmente. Aplicado no Flexbox Froggy.
  • flex-wrap: Quebra linha quando necessário. Essencial no Flexbox Zombies.
  • flex-grow: Faz um item crescer para ocupar espaço. Utilizado no Flexbox Zombies.

Exemplo Real:

Para um portfólio, usaria o Flexbox para organizar cards responsivos, com flex-wrap para quebra de linha e justify-content para distribuir o espaço entre os cards.
